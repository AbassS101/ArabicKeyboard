# Step-by-Step Installation Guide

## How to Install the Arabic Keyboard Extension

### Step 1: Open Chrome Extensions Page
1. Open Google Chrome browser
2. In the address bar, type: `chrome://extensions/`
3. Press **Enter**

### Step 2: Enable Developer Mode
1. Look at the **top right corner** of the extensions page
2. You'll see a toggle switch labeled **"Developer mode"**
3. **Turn it ON** (it should turn blue/highlighted)

### Step 3: Load the Extension
1. After enabling Developer mode, you'll see new buttons appear at the top
2. Click the button that says **"Load unpacked"** (it's usually on the left side)
3. A file browser window will open

### Step 4: Select the Extension Folder
1. In the file browser, navigate to: `C:\Users\aweso\Documents\ArabicKeyboard`
2. **Select the folder** (click on it once to highlight it)
3. Click **"Select Folder"** or **"OK"** button

### Step 5: Verify Installation
1. You should now see "Arabic Keyboard Visual" appear in your extensions list
2. You should see an extension icon appear in your Chrome toolbar (top right, near the address bar)
3. If you don't see the icon, click the puzzle piece icon (ðŸ§©) in the toolbar to see all extensions

### Step 6: Test It Out!
1. Go to any website (like Google, Facebook, etc.)
2. Press **Ctrl + Shift + A** (all three keys together)
3. The Arabic keyboard overlay should appear!
4. Press **Ctrl + Shift + A** again to hide it

## Alternative: Using the Extension Button

1. Click the extension icon in your Chrome toolbar
2. Click the **"Toggle Keyboard"** button
3. The keyboard overlay will appear

## Troubleshooting

**Problem: "Load unpacked" button is grayed out**
- Make sure Developer mode is turned ON

**Problem: Extension doesn't appear**
- Make sure you selected the correct folder (ArabicKeyboard)
- Check that all files are in the folder (manifest.json, content.js, etc.)

**Problem: Keyboard shortcut doesn't work**
- Make sure you're on a webpage (not chrome:// pages)
- Try clicking the extension icon and using the button instead
- The shortcut is: **Ctrl + Shift + A** (hold all three keys)

**Problem: Can't see the extension icon**
- Click the puzzle piece icon (ðŸ§©) in Chrome toolbar
- Find "Arabic Keyboard Visual" and click the pin icon to pin it to toolbar
