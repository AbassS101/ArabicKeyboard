ICON FILES NEEDED
==================

The extension needs three icon files in the icons/ folder:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)  
- icon128.png (128x128 pixels)

OPTION 1: Use the HTML generator
---------------------------------
1. Open create_icons.html in your browser
2. Click "Download Icons"
3. Move the downloaded files to the icons/ folder

OPTION 2: Create your own
-------------------------
Use any image editor to create icons with:
- Background: Purple gradient (#667eea to #764ba2)
- Symbol: Keyboard emoji or Arabic letter
- Size: 16x16, 48x48, and 128x128 pixels

OPTION 3: Use placeholder icons
--------------------------------
The extension will work without icons - Chrome will just show a default icon.
You can add icons later if you want a custom icon for your extension.
