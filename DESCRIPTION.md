# Arabic Keyboard Visual - Extension Description

## Short Description (for Chrome Web Store)
A floating Arabic keyboard overlay that helps you type Arabic characters anywhere. Shows which Arabic letter each key produces with visual key highlighting.

## Full Description

**Arabic Keyboard Visual** is a powerful Chrome extension designed to help you type Arabic text more easily. It displays a floating, interactive Arabic keyboard overlay on any webpage, making it simple to learn and reference the Arabic keyboard layout while typing.

### Key Features:

**Visual Reference**
- See the complete Arabic keyboard layout at a glance
- Each key shows both the English letter and its Arabic equivalent
- Keys highlight when you press them on your physical keyboard

**Fully Customizable**
- Drag the keyboard anywhere on your screen
- Resize it using the +/- buttons or drag the corner handle
- Adjustable size from 60% to 150% of original
- Dark, masculine theme that's easy on the eyes

**Easy to Use**
- Toggle with keyboard shortcut: **Ctrl+Shift+A** (or Cmd+Shift+A on Mac)
- Click the extension icon for quick access
- Press **Escape** to close
- Works on all websites including YouTube, social media, and text editors

**Smart & Persistent**
- Remembers your preferred position and size
- Saves your settings automatically
- Stays on top of videos and other content
- Lightweight and non-intrusive

### Perfect For:
- Learning the Arabic keyboard layout
- Quick reference while typing Arabic
- Students studying Arabic language
- Anyone who needs to type Arabic occasionally
- People switching between English and Arabic keyboards

### How It Works:
1. Install the extension
2. Navigate to any website
3. Press **Ctrl+Shift+A** to show the keyboard
4. Start typing - the keys will highlight as you press them
5. Drag it to your preferred position
6. Resize it to your comfort level

The extension uses the standard Arabic keyboard layout mapping (Q = ض, W = ص, E = ث, etc.) and works seamlessly with your existing keyboard input method.

---

**Note:** This extension is a visual reference tool. You still need to have Arabic input enabled in your system settings to actually type Arabic characters.
